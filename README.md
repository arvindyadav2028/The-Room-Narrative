# 🚦 Parabit INSS – Intelligent Navigation Sentinal System

A Java-based **real-time crowd and vehicle monitoring dashboard** built for large-scale events and smart city use cases.  
This system visualizes live entry data from checkpoints and helps administrators monitor crowd flow and vehicle movement in real time.

Developed as part of a **Smart Mobility & Access Management** solution for mass gatherings and urban safety.

## 📌 Features
- 📊 Live Line Charts (real-time crowd entry)
- 🚗 Vehicle Entry Monitoring
- 🏛️ Checkpoint-Based Filtering (default CpNo = 2)
- 👥 Employee Hierarchy Management
- 🔐 Admin Dashboard Panel
- 🗄️ MySQL Database Integration
- ⚡ JavaFX + JFreeChart UI

## 🛠️ Tech Stack
Frontend: JavaFX  
Charts: JFreeChart  
Backend: Java (JDBC)  
Database: MySQL  
IDE: NetBeans  

## 💻 System Requirements
- Java JDK 8+
- NetBeans 12+
- MySQL Server

## ▶️ How to Run
```bash
git clone https://github.com/shyalu02/PARABITINSS.git
cd ParabitINSS
```

Open in NetBeans → Configure DB → Run Main class(PARABITINSS).

## 📦 Create Executable JAR
Right click project → Clean & Build  
Run:
```bash
java -jar dist/ParabitINSS.jar
```
